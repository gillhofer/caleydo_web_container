Caleydo Web - Server API

Meta

  • about - Returns information about the server.

    Examples

    • about

Data

  • dataset - Returns a list of datasets.

    Examples

    • /api/dataset

  • dataset/id - Returns all rows contained in dataset id.

    Parameters

    • start - Limit returned rows to those >= start. Rows are 0-indexed.
    • end - Limit returned rows to those < end. Rows are 0-indexed.

    Examples

    • /api/dataset/0?start=3&end=5
